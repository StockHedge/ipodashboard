# IPO 공모주 대시보드 — 실무진 설명서

> 민트투자자문 내부 공유용. **API 키 포함 패키지** — 외부 유출 금지.
> 최종 갱신: 2026-06-19 (락업 관리 페이지 · 토스 성과 · 라이브 상장 · 개인 청약경쟁률 반영)

---

## ⚠️ 보안 (가장 먼저)

이 패키지의 **`.env` 파일에는 실제 API 키가 들어 있습니다**:

- `DART_API_KEY` — 전자공시(증권신고서)
- `KIS_APP_KEY` / `KIS_APP_SECRET` — 한국투자증권 (모의투자 모드, 종목 실시간 시세)
- `KRX_API_KEY` — 한국거래소 OpenAPI (지수 일별시세, **활성**)
- `NAVER_NEWS_CLIENT_ID` / `NAVER_NEWS_CLIENT_SECRET` — 네이버 뉴스 검색
- `TOSS_APP_KEY` / `TOSS_APP_SECRET` — 토스증권 Open API (상장후 일봉 성과)

**금지 사항**

- 이 ZIP / `.env` 를 외부(메신저·이메일·클라우드 공개 링크)로 전달 금지
- GitHub 등 공개 저장소 커밋 금지 (`.gitignore` 에 `.env` 등록되어 있으나 ZIP 자체 관리 주의)
- 키 유출 의심 시 즉시 각 콘솔(opendart / KIS / KRX / 네이버 / 토스)에서 **폐기·재발급**

키는 **서버(serve.py) 에서만** 사용되며 브라우저로 노출되지 않습니다(모든 외부 호출은 로컬/서버 프록시 경유). 토스 키는 IP 허용목록 방식이라 **등록된 IP(사무실/특정 PC)에서만** 작동합니다(아래 6절).

## 1. 두 가지 사용 방법

### A. 배포 서버 (실무진 일상 사용 — 설치 불필요)

브라우저에서 **https://ipo-dashboard-5qbx.onrender.com** 접속 → Basic Auth 비밀번호 입력.
- 24시간 접속 가능(무료 호스팅이라 미사용 시 슬립 → 첫 접속 30초 정도 로딩될 수 있음).
- 신규 상장·뉴스·일정은 서버가 자동 갱신(30분 캐시).

### B. 로컬 실행 (개발·데이터 갱신용)

```powershell
# Python 3.10+ 필요
cd ipo-dashboard
pip install -r requirements-server.txt   # 서버 실행용 (requests, python-dotenv, finance-datareader)
python scripts/serve.py
```
브라우저에서 **http://localhost:8000** 접속.

## 4. 데이터 갱신

- **신규 상장·뉴스·일정**: 자동(서버 30분 캐시). 별도 작업 불필요.
- **신규 종목 상장후 성과**(고가·기간수익률): 토스 배치 — 아래 6절.
- **기준 데이터 시트 교체 시**: `python scripts/extract_ipo_data.py` 로 재생성 후 `refresh_baseline_perf.py` 로 성과 덮기.

1. **락업 관리 페이지 신설** — 전 종목 × 4구간(상장일+15/30/90/180일) 해제 일정. 다가오는 순 정렬, 사이드바 D-7 임박 배지, 구간/기간/참여 필터.

## 6. 토스 성과 배치 실행 (등록 IP에서만)

토스 Open API는 **호출 PC의 공인 IP를 토스 개발자센터에 등록**해야 작동합니다(미등록 시 `403 IP not allowed`).

```powershell
# 신규 상장 종목 성과 → data/recent-perf.json
python scripts/fetch_prices.py

# 기준 228건 성과 재산출 → data/ipo-recent.js (약 1분)
python scripts/refresh_baseline_perf.py
```

생성된 정적 파일(`recent-perf.json`, `ipo-recent.js`)을 커밋·배포하면, 배포 서버는 토스를 직접 호출하지 않고 이 파일만 읽어 성과를 표시합니다(배포 환경은 토스 IP 미등록이라 직접 호출 불가).

> IP가 바뀌면(동적 IP) 토스 개발자센터에 재등록 후 실행하세요.

## 7. 자주 묻는 질문

- **마켓 시세 "미수신"**: serve.py 실행 여부 확인. `http://localhost:8000/api/ping` 진단.
- **배포 서버 첫 접속이 느림**: 무료 호스팅 슬립 → 깨어나는 데 ~30초. 정상.
- **종목명 옆 ↗ 안 보임**: ticker 누락 종목 → 네이버 "검색" 으로 자동 fallback.
- **신규 종목 성과(고가·6M)가 비어 있음**: 토스 배치(6절) 미실행 또는 상장 6개월 미경과(3M/6M은 시간 경과 후 채워짐).
- **스팩 성과 없음**: 스팩은 토스 ticker 비표준/합병 전 가격 고정이라 성과 산출 대상 외.
